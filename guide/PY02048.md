# PY02048 - Tách nhóm tối ưu

## Đề bài
Cho dãy số `A` có `N` phần tử là các số nguyên dương. Cần tách dãy số thành ít nhất bao nhiêu nhóm sao cho mỗi số trong nhóm đều có chênh lệch với ít nhất một số khác cùng nhóm **không vượt quá `K`** (`<= K`).
In ra số lượng nhóm ít nhất có thể.

## Ý tưởng cốt lõi
Bài toán bị làm phức tạp hóa bởi ngôn từ. Bản chất để có số nhóm ít nhất: Hãy nhóm tất cả những thằng có khoảng cách gần nhau vào chung 1 nhóm.
1. **Sort mảng:** Sắp xếp mảng tăng dần để các số gần nhau đứng cạnh nhau.
2. **Kỹ thuật Đếm Nhát Cắt:** Đi dọc theo mảng, nếu phát hiện 2 số liền kề cách nhau một khoảng `> K` (Đứt xích), ta bắt buộc phải chia cắt chúng bằng một "nhát cắt".
3. **Quy luật:** `Số Nhóm = Số Nhát Cắt + 1`.

---

## Pseudocode

### Luồng xử lý chính
```text
Bắt đầu:
    Đọc số N và K
    Đọc mảng A
    Sắp xếp mảng A (A.sort)
    
    Khởi tạo đếm nhát cắt c = 0
    
    Lặp i chạy từ 0 đến len(A) - 2:
        Nếu A[i+1] - A[i] > K:  // Đứt xích
            Tăng c thêm 1       // Vung một nhát cắt
            
    In ra c + 1                 // Số nhóm luôn bằng số nhát cắt + 1
```

---

## Pythonic Way (Phương pháp tối ưu)
Chỉ cần dùng vòng lặp `for` thông thường quét qua mảng. Không cần cắt mảng vật lý vì đề chỉ yêu cầu in ra số lượng.

```python
n, k = map(int, input().split())
a = list(map(int, input().split()))

a.sort()
c = 0
for i in range(len(a) - 1):
    if a[i+1] - a[i] > k:
        c += 1

print(c + 1)
```

**Mở rộng:** Nếu đề yêu cầu IN RA CÁC MẢNG bị cắt, ta áp dụng kỹ thuật **Hai Con Trỏ (Anchor & Scan)** kết hợp **Xả bộ đệm (Buffer Flush)**.
```python
start = 0  # Mỏ neo
for i in range(len(a) - 1):
    if a[i+1] - a[i] > k:
        print("Nhóm:", a[start : i+1]) # Cắt từ mỏ neo đến vết đứt
        start = i + 1                  # Cắm mỏ neo sang vị trí mới

# Xả nốt khúc cuối cùng còn đọng lại
print("Nhóm:", a[start : len(a)])
```
