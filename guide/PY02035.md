## Ý tưởng cốt lõi
1. Tách chuỗi thành các cụm 2 chữ số (bước nhảy 2) và nạp vào mảng `a` (bắt buộc dùng List, không dùng `set` để bảo toàn quân số phục vụ việc đếm).
2. Lọc trùng mảng `a` thành mảng `b` để tạo danh sách đối chiếu. Dùng kỹ thuật `dict.fromkeys(a)` để lọc trùng mà không làm xáo trộn thứ tự gốc. (Lưu ý: Nếu đề yêu cầu in tăng dần thì thêm hàm `b.sort()`).
3. Đếm tần suất: Duyệt từng phần tử của mảng `b`, quét lại mảng `a` để đếm số lần xuất hiện. Đối chiếu với ngưỡng `k`, nếu đạt chuẩn thì in ra.
4. Xử lý "NOT FOUND": Bắt buộc sử dụng biến cờ (Flag) `find = False`. Chỉ khi in ra thành công mới đổi cờ thành `True`. Nếu quét cạn toàn bộ mà cờ vẫn `False` thì mới chốt hạ in `"NOT FOUND"`. Tuyệt đối không dùng `else` dính với vòng `for` nếu không có cơ chế `break`.

## Pseudocode

```text
Bắt đầu (Main):
    Nhập chuỗi s
    Nhập ngưỡng tối thiểu k
    
    // Bước 1: Cắt chuỗi 
    Khởi tạo mảng a = []
    Cho i chạy từ 0 đến len(s) - 2, bước nhảy là 2:
        Cắt s[i : i+2] và thêm vào mảng a
        
    // Bước 2: Lọc trùng tạo danh sách tướng lĩnh
    Khởi tạo mảng b = list(dict.fromkeys(a))
    
    // Bước 3: Kiểm kê quân số
    find = False
    Cho mỗi phần tử i trong b:
        Khởi tạo đếm c = 0
        Cho mỗi phần tử j trong a:
            Nếu i == j:
                c = c + 1
                
        Nếu c >= k:
            find = True
            In ra i và c (ngăn cách bởi khoảng trắng)
            
    // Bước 4: Xử lý ngoại lệ
    Nếu find == False:
        In ra "NOT FOUND"
```

## Pythonic Way (Phương pháp tối ưu)
Bài toán "đếm theo thứ tự tăng dần" là sân khấu để `collections.Counter` tỏa sáng kết hợp với tính năng duyệt key của dictionary.

```python
from collections import Counter

s = input()
k = int(input())

# Cắt chuỗi và nạp vào mảng a
a = [int(s[i:i+2]) for i in range(0, len(s) - 1, 2)]

# Đếm tần suất
dem = Counter(a)

find = False

# Sắp xếp key (danh sách tướng lĩnh) tăng dần và duyệt
for so in sorted(dem.keys()):
    if dem[so] >= k:
        print(so, dem[so])
        find = True

if not find:
    print("NOT FOUND")
```

**Ưu điểm:** Khử sạch vòng lặp lồng $O(N^2)$ của thuật toán đếm thủ công. Chạy cực nhanh $O(N \log N)$ (do phải sort). Code thanh lịch và không rườm rà.
