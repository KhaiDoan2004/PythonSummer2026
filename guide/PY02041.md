# PY02041 - Tổng số nguyên lớn

## Đề bài
Cho hai xâu ký tự A và B mô tả hai số nguyên dương lớn có thể có đến 1000 chữ số.
Tính tổng A + B.

## Ý tưởng cốt lõi
1. Đọc hai chuỗi A và B từ Input theo từng dòng.
2. Ép kiểu trực tiếp hai chuỗi này sang số nguyên (`int()`). 
3. Cộng hai số và in ra màn hình.

*(Lưu ý: Không giống C/C++ bị giới hạn bởi kiểu dữ liệu `long long` hay Java phải dùng đối tượng `BigInteger` phức tạp, lõi của Python tự động hỗ trợ số nguyên lớn với độ dài vô hạn - chỉ phụ thuộc vào dung lượng RAM. Do đó, việc cộng hai số 1000 chữ số bằng Python cũng đơn giản nhẹ nhàng hệt như cộng 1 + 1).*

### 1. Luồng xử lý chính

```text
Bắt đầu:
    Đọc chuỗi s từ dòng 1

    // Ép kiểu sang số nguyên (Python tự động lo phần tràn bộ nhớ)

```

## Pythonic Way (Phương pháp tối ưu)
Code của Chủ nhân đã mang đúng tinh thần Pythonic nhất rồi! Chúng ta chỉ việc lược bỏ các biến trung gian để gộp nó thành 1 dòng code duy nhất cực kỳ "ngầu".

```python
print(int(input()) + int(input()))
```

**Ưu điểm:** Tận dụng tối đa sức mạnh BigInt có sẵn của Python. Viết một dòng giúp tiết kiệm bộ nhớ, không tạo ra các biến trung gian dư thừa.
