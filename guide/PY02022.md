# PY02022 - Liệt kê số nguyên tố trong dãy

## Đề bài
Cho dãy số nguyên `A` có `N` phần tử. Hãy đếm số lần xuất hiện của các số nguyên tố trong dãy và in ra theo đúng thứ tự xuất hiện lần đầu tiên của chúng.

## Ý tưởng cốt lõi
1. **Lọc nguyên tố & Lọc trùng giữ thứ tự:** Duyệt qua mảng `A`. Nếu một số là số nguyên tố và CHƯA CÓ mặt trong mảng kết quả `B` (dùng `not in B`), ta mới thêm nó vào `B`. Thao tác này giúp `B` chứa các số nguyên tố duy nhất, bảo toàn được thứ tự xuất hiện ban đầu.
2. **Đếm tần suất:** Duyệt qua từng số trong mảng `B` vừa lập. Đối với mỗi số, ta quét lại mảng gốc `A` xem nó xuất hiện bao nhiêu lần, đếm và in ra kết quả.

## Pseudocode

### Luồng xử lý chính
```text
Bắt đầu:
    Đọc N và mảng A
    Khởi tạo mảng rỗng B
    
    // Bước 1: Lọc số nguyên tố duy nhất
    Lặp i từ đầu đến cuối mảng A:
        Nếu A[i] là số nguyên tố VÀ A[i] chưa nằm trong mảng B:
            Nhét A[i] vào B
            
    // Bước 2: Đếm và In
    Lặp i từ đầu đến cuối mảng B:
        Khởi tạo biến đếm c = 0
        
        Lặp j từ đầu đến cuối mảng A:
            Nếu A[j] bằng với B[i]:
                Tăng biến đếm c thêm 1
                
        In ra B[i] và c
```

## Pythonic Way (Phương pháp tối ưu)
Cách của Chủ nhân làm cực kỳ logic và an toàn. Tuy nhiên, nếu muốn tận dụng triệt để "đồ chơi" có sẵn của Python để code ngắn gọn hơn, ta có thể dùng `list(dict.fromkeys())` để lọc trùng và hàm `.count()` để đếm mà không cần tự viết vòng lặp lồng nhau (tránh vòng lặp lồng vòng lặp $O(N^2)$).

```python
import math

def checksonguyento(n):
    if n < 2: return False
    for i in range(2, math.isqrt(n) + 1):
        if n % i == 0: return False
    return True

n = int(input())
a = list(map(int, input().split()))

# 1. Nhặt ra TẤT CẢ các số nguyên tố (có thể bị trùng)
primes = [x for x in a if checksonguyento(x)]

# 2. Lọc trùng nhưng vẫn giữ nguyên thứ tự xuất hiện
b = list(dict.fromkeys(primes))

# 3. Duyệt mảng đã lọc và dùng hàm count() để đếm luôn
for x in b:
    print(x, primes.count(x))
```

**Bài học rút ra:** 
Dòng lệnh huyền thoại `x not in b` để kiểm tra tồn tại là cực kỳ hữu dụng, nhưng nó luôn chạy ngầm một vòng lặp `for` bên dưới. Ngài hãy cất kỹ kỹ thuật tự cắm cờ `for-else` vào đầu để lôi ra dùng nếu lỡ đi thi ngôn ngữ khác nhé!
