# PY01055 - Số xen kẽ

## Đề bài
Kiểm tra một số nguyên có phải là **số xen kẽ** hay không. 
Một số được coi là xen kẽ nếu thoả mãn 3 điều kiện:
1. Số lượng chữ số của số đó là số lẻ.
2. Chữ số đầu tiên phải khác chữ số thứ hai.
3. Tất cả các chữ số ở vị trí chẵn (tính từ chỉ số 0) phải giống nhau.

## Ý tưởng cốt lõi
1. Đọc đầu vào dạng chuỗi để dễ dàng truy xuất từng ký tự qua chỉ số (index).
2. Sàng lọc loại ngay: Nếu độ dài chuỗi chẵn `len(s) % 2 == 0` hoặc ký tự đầu tiên bằng ký tự thứ hai `s[0] == s[1]`, ta kết luận là sai và in "NO" ngay.
3. Duyệt để kiểm tra: Nếu qua được vòng kiểm duyệt trên, ta chỉ cần xét các ký tự ở vị trí chẵn (0, 2, 4...) xem chúng có bằng nhau hay không. Bằng cách dùng bước nhảy là 2 trong vòng lặp và so sánh `s[i]` với `s[i+2]`. Nếu phát hiện khác nhau, lập tức ngắt vòng lặp (break) và in "NO".
4. Nếu vượt qua tất cả không lỗi lầm gì, in "YES".

```text
Lặp số_lượng_test lần:

            Nếu s[i] != s[i+2]:

                Dừng Lặp (break)
            
    Nếu check là ĐÚNG:

```

## Giải pháp thực tế (Dựa trên tư duy của Chủ công)
Đây là đoạn mã kết hợp tư duy logic `if/else` tường minh, đảm bảo không sót trường hợp nào và dùng cờ `check` chuẩn bài.

```python
n = int(input())
for _ in range(n):
    s = input().strip()
    
    check = True # Mặc định tin tưởng chuỗi là số xen kẽ
    
    # Sàng lọc điều kiện loại trừ
    if len(s) % 2 == 0 or s[0] == s[1]:
        check = False
    else:
        # Nếu không dính lỗi, duyệt các vị trí chẵn
        for i in range(0, len(s) - 2, 2):
            if s[i] != s[i+2]:
                check = False
                break
                
    if check:
        print("YES")
    else:
        print("NO")
```
