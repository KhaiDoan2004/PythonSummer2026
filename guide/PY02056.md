# PY02056 - Số thuận nghịch lớn nhất trong ma trận

## Đề bài
Tìm số thuận nghịch lớn nhất trong ma trận và in ra các vị trí xuất hiện của nó. Số thuận nghịch hợp lệ phải có ít nhất 2 chữ số (>= 10).

## Ý tưởng cốt lõi
1. Viết hàm kiểm tra một số có phải số thuận nghịch hay không. Điều kiện bắt buộc là số đó phải >= 10. Đảo ngược số đó và so sánh với giá trị ban đầu.
2. Duyệt qua toàn bộ các phần tử của ma trận, dùng biến `max_val` để tìm giá trị thuận nghịch lớn nhất.
3. Nếu `max_val` vẫn bằng 0 (không tìm thấy số nào), in ra "NOT FOUND".
4. Nếu có tìm thấy, in ra giá trị đó. Sau đó duyệt lại ma trận một lần nữa, nếu phần tử nào bằng `max_val` thì in ra vị trí của nó (định dạng `Vi tri [i][j]`).

---

## Pseudocode
```text
### 1. Hàm kiểm tra số thuận nghịch
Hàm checksothuannghich(n):
    Nếu n < 10:
        Trả về Sai (False)
    Lưu giá trị gốc o_n = n
    Khởi tạo r = 0
    Chừng nào n > 0:
        r = r * 10 + (n chia dư cho 10)
        n = n chia nguyên cho 10
    Nếu r bằng o_n:
        Trả về Đúng (True)
    Trả về Sai (False)

### 2. Luồng xử lý chính
Nhập n (số hàng) và m (số cột)
Khởi tạo ma trận a = rỗng
Đọc dữ liệu ma trận từ đầu vào và lưu vào a

Khởi tạo max_val = 0
Duyệt i từ 0 đến n-1:
    Duyệt j từ 0 đến m-1:
        Nếu checksothuannghich(a[i][j]) là Đúng:
            Nếu a[i][j] > max_val:
                max_val = a[i][j]

Nếu max_val == 0:
    In ra "NOT FOUND"
Ngược lại:
    In ra max_val
    Duyệt i từ 0 đến n-1:
        Duyệt j từ 0 đến m-1:
            Nếu a[i][j] == max_val:
                In ra "Vi tri [i][j]"
```

## Pythonic Way (Phương pháp tối ưu)
Dùng hàm `str()` để kiểm tra số thuận nghịch cực nhanh thay vì phải dùng vòng lặp chia số.

```python
def is_palindrome(n):
    s = str(n)
    # Kiểm tra số có từ 2 chữ số trở lên và là số thuận nghịch
    return len(s) > 1 and s == s[::-1]

n, m = map(int, input().split())
a = [list(map(int, input().split())) for _ in range(n)]

max_val = 0
for row in a:
    for val in row:
        if is_palindrome(val) and val > max_val:
            max_val = val

if max_val == 0:
    print("NOT FOUND")
else:
    print(max_val)
    for i in range(n):
        for j in range(m):
            if a[i][j] == max_val:
                print(f"Vi tri [{i}][{j}]")
```
Ưu điểm:
- Mã nguồn gọn gàng, sử dụng list comprehension để nhập liệu ma trận ngắn gọn.
- `s == s[::-1]` tận dụng cách cắt chuỗi (slicing) tối ưu của Python, chạy nhanh hơn nhiều so với việc chia lấy dư từng số.
