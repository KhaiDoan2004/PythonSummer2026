## Ý tưởng cốt lõi
1. Tách chuỗi gốc thành các cụm 2 ký tự bằng vòng lặp bước nhảy 2, lưu toàn bộ vào mảng `a` (mảng chứa toàn quân).
2. Lọc ra mảng `b` (danh sách tướng lĩnh) bằng cách duyệt qua `a` và kiểm tra `not in`. Mảng `b` này giữ nguyên thứ tự xuất hiện đầu tiên của mỗi phần tử.
3. Đếm số lần xuất hiện bằng thuật toán vét cạn (vòng lặp kép): Duyệt từng phần tử của mảng `b`, với mỗi phần tử ta quét lại toàn bộ mảng `a` để đếm tần suất xuất hiện và in kết quả.

---

## Pseudocode

```text
Bắt đầu (Main):
    Nhập chuỗi s
    Khởi tạo mảng a = []
    
    // Bước 1: Cắt chuỗi 
    Cho i chạy từ 0 đến len(s) - 2, bước nhảy là 2:
        Cắt s[i : i+2] và thêm vào a
        
    // Bước 2: Lọc trùng
    Khởi tạo mảng b = []
    Cho mỗi phần tử x trong a:
        Nếu x chưa có trong b:
            Thêm x vào b
            
    // Bước 3: Đếm tần suất và In kết quả
    Cho mỗi phần tử x trong b:
        Khởi tạo biến đếm c = 0
        Cho mỗi phần tử y trong a:
            Nếu x == y:
                c = c + 1
        In ra x và c trên cùng một dòng
```

## Pythonic Way (Phương pháp tối ưu)
Bài toán đếm số lượng đích thị là sân nhà của `collections.Counter`. Kể từ Python 3.7, dictionary (bao gồm cả Counter) đã mặc định lưu giữ nguyên thứ tự xuất hiện.

```python
from collections import Counter

s = input()

# Cắt chuỗi và nạp vào mảng a
a = [int(s[i:i+2]) for i in range(0, len(s) - 1, 2)]

# Đếm và giữ nguyên thứ tự
dem = Counter(a)

for k, v in dem.items():
    print(k, v)
```

**Ưu điểm:** Không cần thuật toán vét cạn 2 vòng lặp ($O(N^2)$). `Counter` giúp giải quyết bài toán chỉ trong $O(N)$ thời gian với số dòng code cực ít.
