## Ý tưởng cốt lõi
1. Đọc và nạp mảng `A` trên cùng 1 dòng bằng `list(map(int, input().split()))`.
2. Lọc mảng không trùng lặp và giữ nguyên thứ tự bằng cách khởi tạo mảng `B` rỗng, duyệt mảng `A`, nếu phần tử chưa có trong `B` thì `append`.
3. Tìm vị trí phân chia: Sử dụng thuật toán vét cạn (brute-force). Tại mỗi vị trí `i`, biến `c` cộng tích lũy tổng trái, biến `d` dùng vòng lặp con duyệt phần còn lại để tính tổng phải.
4. Điều kiện dừng: Dùng hàm kiểm tra số nguyên tố cho cả `c` và `d`. Sử dụng biến cờ `find` (boolean flag) để kiểm soát việc in ra "NOT FOUND".

## Pythonic Way (Phương pháp tối ưu)
Để tránh vòng lặp lồng nhau ($O(N^2)$) khi tính tổng hai nửa, ta có thể tính trước tổng toàn bộ mảng. Sau đó khi duyệt qua từng vị trí, ta chỉ việc cộng dồn vào nửa trái, nửa phải sẽ bằng tổng trừ đi nửa trái. Độ phức tạp giảm xuống $O(N)$.

```python
import math

def is_prime(n):
    if n < 2: return False
    for i in range(2, math.isqrt(n) + 1):
        if n % i == 0: return False
    return True

n = int(input())
a = list(map(int, input().split()))

# Lọc trùng giữ nguyên thứ tự
b = list(dict.fromkeys(a))

total_sum = sum(b)
left_sum = 0
find = False

for i in range(len(b)):
    left_sum += b[i]
    right_sum = total_sum - left_sum
    
    if is_prime(left_sum) and is_prime(right_sum):
        print(i)
        find = True
        break

if not find:
    print("NOT FOUND")
```

**Ưu điểm:** Thuật toán Prefix Sum cơ bản giúp giảm độ phức tạp từ $O(N^2)$ xuống $O(N)$. Dùng `dict.fromkeys` lọc trùng ngắn gọn 1 dòng.
