## Ý tưởng cốt lõi
1. Tách biệt logic: Viết riêng hàm `check_so_nguyen_to(n)` tối ưu bằng cách duyệt đến căn bậc 2 (`math.isqrt(n)`).
2. Nhập ma trận: Đọc kích thước $N \times M$, dùng vòng lặp đọc từng hàng rồi nạp vào mảng 2 chiều.
3. Duyệt ma trận: Dùng 2 vòng lặp `for` lồng nhau đi qua từng tọa độ `(i, j)`.
4. Biến đổi: 
   - Nếu `matrix[i][j]` là số nguyên tố $\rightarrow$ Gán lại bằng $1$.
   - Ngược lại $\rightarrow$ Gán lại bằng $0$.
5. Xuất kết quả: Duyệt từng hàng và in các phần tử cách nhau bởi dấu cách.

## Pseudocode

### 1. Hàm kiểm tra nguyên tố
```text
Hàm check_so_nguyen_to(n):
    Nếu n < 2:
        Trả về False
    Cho i chạy từ 2 đến Căn_Bậc_2_Của(n):
        Nếu n chia hết cho i:
            Trả về False
    Trả về True
```

### 2. Luồng xử lý chính (Main)
```text
Bắt đầu:
    Nhập n, m từ cùng 1 dòng
    Khởi tạo matrix = []
    
    // Đọc ma trận
    Lặp n lần:
        Đọc danh sách số trên dòng -> hang
        Nạp hang vào matrix
        
    // Duyệt và biến đổi
    Lặp i chạy từ 0 đến n - 1:
        Lặp j chạy từ 0 đến m - 1:
            Nếu check_so_nguyen_to(matrix[i][j]) là True:
                matrix[i][j] = 1
            Ngược lại:
                matrix[i][j] = 0
                
    // In kết quả
    Lặp qua từng hang trong matrix:
        In ra toàn bộ hang (ngăn cách bởi dấu cách)
```

## Pythonic Way (Phương pháp tối ưu)
Sử dụng **List Comprehension** (Trình biên dịch mảng) để rút gọn 2 vòng lặp `for` cồng kềnh thành một dòng code duy nhất cực kỳ thanh lịch.

```python
import math

def is_prime(n):
    if n < 2: return False
    for i in range(2, math.isqrt(n) + 1):
        if n % i == 0: return False
    return True

n, m = map(int, input().split())

for _ in range(n):
    # Đọc từng hàng, biến đổi thành 1/0 và in ngay lập tức
    row = list(map(int, input().split()))
    result = [1 if is_prime(x) else 0 for x in row]
    print(*result)
```

**Ưu điểm:**
- Không cần tốn bộ nhớ để khởi tạo mảng 2 chiều khổng lồ rồi mới in ra, đọc dòng nào xử lý in luôn dòng đó.
- Dấu `*result` (unpacking) tự động bung mảng thành các phần tử cách nhau bởi dấu cách, không cần vòng lặp in thủ công.
