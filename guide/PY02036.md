## Ý tưởng cốt lõi
1. **Khái niệm:** Hai số được gọi là "nguyên tố cùng nhau" (Coprime) khi và chỉ khi Ước chung lớn nhất (GCD) của chúng bằng 1. Chúng không nhất thiết phải là số nguyên tố. Python hỗ trợ tận răng với hàm `math.gcd()`.
2. **Nhập dữ liệu:** Tránh bẫy Input. Mảng được ghi trên cùng 1 dòng, dùng `list(map(int, input().split()))` để đọc trọn gói, tuyệt đối không dùng vòng lặp `for` để bắt đọc từng dòng.
3. **Sắp xếp mảng:** Đề yêu cầu in kết quả theo thứ tự tăng dần, nên phải gọi `a.sort()` ngay từ đầu để ép toàn quân xếp hàng.
4. **Bắt cặp Tổ hợp ($C^2_N$):** Bắt buộc dùng vị trí (Index) để vét cạn. Vòng lặp ngoài `i` quét toàn mảng, vòng lặp trong `j` luôn chạy từ `i + 1`. Cách này chặn đứng 100% lỗi tự chém nhau (ghép 3 với 3) và lỗi lặp ngược (in cả 3-7 và 7-3).

## Pseudocode

```text
Bắt đầu (Main):
    Import thư viện math
    Nhập n
    Nhập mảng a (tự động cắt và ép kiểu nguyên)
    
    // Bước 1: Xếp đội hình tăng dần
    Gọi hàm a.sort()
    
    // Bước 2: Vét cạn bắt cặp (Không trùng lặp, không lùi bước)
    Cho i chạy từ 0 đến len(a) - 1:
        Cho j chạy từ i + 1 đến len(a) - 1:
        
            // Bước 3: Phán xét
            Nếu math.gcd(a[i], a[j]) == 1:
                In ra cặp a[i] và a[j] cách nhau bởi khoảng trắng
```

## Pythonic Way (Phương pháp tối ưu)
Python cung cấp thư viện `itertools.combinations` sinh ra để sinh tổ hợp một cách chuyên nghiệp và tối ưu, thay thế hoàn toàn thuật toán vét cạn vòng lặp kép.

```python
import math
from itertools import combinations

n = int(input())
a = list(map(int, input().split()))

a.sort() # Sắp xếp tăng dần ngay từ đầu

# Bắt cặp 2 phần tử mọi khả năng có thể
for x, y in combinations(a, 2):
    if math.gcd(x, y) == 1:
        print(x, y)
```

**Ưu điểm:** Không cần bận tâm về kiểm soát index bằng các vòng lặp lồng nhau (tránh lỗi ngớ ngẩn do vòng lặp sai biên). Code rất đẹp và đậm chất toán học.
