# PY02032 - Liệt kê các số có hai chữ số tăng dần

## Đề bài
<!-- Thả ảnh đề bài vào đây -->

## Ý tưởng cốt lõi
1. Đọc vào chuỗi ký tự liền nhau `s`.
2. Tách chuỗi: Dùng vòng lặp `for` với bước nhảy 2 để cắt ra từng cặp 2 ký tự, ép kiểu số nguyên (`int`) và nạp vào mảng `a`.
3. Lọc trùng: Dùng vòng lặp duyệt mảng `a`, kiểm tra nếu phần tử chưa tồn tại trong mảng `b` thì mới thêm vào `b` (Thuật toán tìm kiếm tuyến tính).
4. Sắp xếp: Dùng thuật toán sắp xếp tăng dần cho mảng `b` (gọi hàm `.sort()`).
5. In kết quả: Duyệt mảng `b` và in các phần tử trên cùng một dòng, cách nhau bởi khoảng trắng.

## Pseudocode

```text
Bắt đầu (Main):
    Nhập chuỗi s
    
    // Bước 1: Tách chuỗi thành mảng số
    Khởi tạo mảng a = []
    Cho i chạy từ 0 đến len(s) - 2, bước nhảy là 2:
        Cắt chuỗi con từ vị trí i đến i+1 (độ dài 2)
        so = ép kiểu số nguyên chuỗi vừa cắt
        Thêm so vào mảng a
        
    // Bước 2: Lọc trùng (không dùng set)
    Khởi tạo mảng b = []
    Cho i chạy từ 0 đến len(a) - 1:
        Nếu a[i] chưa nằm trong b:
            Thêm a[i] vào b
            
    // Bước 3: Sắp xếp
    Sắp xếp mảng b theo thứ tự tăng dần
    
    // Bước 4: In kết quả
    Cho i chạy từ 0 đến len(b) - 1:
        In ra b[i] (thêm khoảng trắng ở đuôi, không xuống dòng)
```

## Pythonic Way (Phương pháp tối ưu)
Thay vì dùng vòng lặp thủ công để duyệt và lọc trùng, ta ứng dụng tính chất của `set()` và `List Comprehension` để gom code lại trong vài dòng ngắn gọn.

```python
s = input()

# Dùng List Comprehension cắt chuỗi, ép kiểu và đưa vào set để khử trùng lặp
a = set(int(s[i:i+2]) for i in range(0, len(s) - 1, 2))

# Sắp xếp và in ra màn hình
print(*sorted(list(a)))
```

**Ưu điểm:** Code sạch, vận dụng đúng thế mạnh của `set` (lọc trùng siêu tốc $O(1)$) và thủ thuật `*` để in danh sách cực nhanh.
