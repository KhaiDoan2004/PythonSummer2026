## Ý tưởng cốt lõi
1. **Bản chất toán học:** Để tìm số lượng cặp có thể ghép được từ $K$ đồng xu, ta dùng Toán Tổ Hợp (Tổ hợp chập 2 của $K$). Công thức thần thánh là: `K * (K - 1) // 2`. Nhờ công thức này, ngài không cần phải vét cạn (lặp lồng nhau) để tìm từng cặp một cách khổ sở.
2. **Quét Hàng ngang:** Quét từng hàng, đếm số lượng chữ `C`. Python cung cấp sẵn hàm `hang.count('C')` để đếm số lượng một ký tự cụ thể trong chuỗi cực kỳ nhanh chóng.
3. **Quét Cột dọc:** Để đếm dọc, ngài cần cố định cột `j` bằng vòng lặp ngoài, và cho hàng `i` trượt từ trên xuống dưới bằng vòng lặp trong. Gặp `C` thì tăng biến đếm lên 1.
4. **Tổng kết:** Tổng số cặp theo hàng và theo cột cộng lại chính là đáp án cuối cùng.

## Pseudocode

```text
Bắt đầu (Main):
    Nhập kích thước lưới: n
    Khởi tạo ma trận rỗng: matrix = []
    
    // Bước 1: Nạp bản đồ
    Lặp n lần:
        Đọc 1 chuỗi ký tự (1 hàng)
        Thêm chuỗi đó vào matrix
        
    Khởi tạo biến đếm tổng: tong = 0
    
    // Bước 2: Càn quét HÀNG NGANG
    Cho mỗi hang trong matrix:
        k = đếm số chữ 'C' trong hang (sử dụng hàm hang.count('C'))
        tong = tong + k * (k - 1) // 2
        
    // Bước 3: Càn quét CỘT DỌC
    Cho cột j chạy từ 0 đến n - 1:
        Khởi tạo k = 0  // Đếm số 'C' cho cột j
        Cho hàng i chạy từ 0 đến n - 1:
            Nếu matrix[i][j] == 'C':
                k = k + 1
                
        tong = tong + k * (k - 1) // 2
        
    // Bước 4: Chốt sổ
    In ra tong
```

## Pythonic Way (Phương pháp tối ưu)
Python có sẵn hàm `math.comb(k, 2)` chuyên dùng để tính tổ hợp chập 2. Kèm theo đó là thủ thuật "Chuyển vị ma trận" (Transpose) siêu việt bằng `zip(*matrix)` để biến việc duyệt cột dọc trở nên dễ như duyệt hàng ngang.

```python
import math

n = int(input())
matrix = [input() for _ in range(n)]

tong = 0

# Càn quét Hàng Ngang
for row in matrix:
    c_count = row.count('C')
    tong += math.comb(c_count, 2)

# Càn quét Cột Dọc (Dùng zip(*matrix) biến cột thành hàng)
for col in zip(*matrix):
    c_count = col.count('C')
    tong += math.comb(c_count, 2)

print(tong)
```

**Ưu điểm:** Triệt tiêu hoàn toàn vòng lặp 2 lớp. Logic xử lý cột giống hệt xử lý hàng nhờ phép chuyển vị thần thánh `zip(*matrix)`. Gọn gàng và mạnh mẽ vô cùng.
