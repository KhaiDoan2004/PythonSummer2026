# PY01004 - Nguyên tố cùng nhau

## Đề bài
Cho số nguyên dương N. Tìm số lượng các số K sao cho 1 <= K <= N và Ước chung lớn nhất (GCD) của K và N bằng 1. Gọi số lượng đó là K_count. Kiểm tra xem K_count có phải là số nguyên tố hay không. In ra `YES` hoặc `NO`.

## Ý tưởng cốt lõi
1. Đề bài có 2 vế rõ ràng: (1) Đếm số nguyên tố cùng nhau, (2) Kiểm tra nguyên tố của chính cái số đếm đó.
2. Xây dựng một hàm `checksonguyento(x)` chuẩn xác (chú ý vòng lặp `math.isqrt(x) + 1`).
3. Dùng hàm tính Ước chung lớn nhất có sẵn của Python `math.gcd(a, b)` để tiết kiệm thời gian.
4. Lặp từ 1 đến N, nếu `math.gcd(i, N) == 1` thì tăng biến đếm lên 1.
5. Sau vòng lặp, kiểm tra biến đếm có phải số nguyên tố không rồi in `YES`/`NO`.

---

## Pseudocode

```text
Hàm checksonguyento(x):
    Nếu x < 2: Trả về Sai
    Lặp i từ 2 đến căn bậc 2 của x:
        Nếu x chia hết cho i:
            Trả về Sai
    Trả về Đúng

Bắt đầu (Main):
    Nhập T (số test case)
    Lặp T lần:
        Nhập N
        Khởi tạo biến đếm = 0
        
        Lặp i từ 1 đến N:
            Nếu UCLN của (i, N) bằng 1:
                Tăng biến đếm lên 1
                
        Nếu checksonguyento(biến đếm) là Đúng:
            In "YES"
        Ngược lại:
            In "NO"
```

## Pythonic Way (Phương pháp tối ưu)
Thay vì tạo mảng `a.append(i)` rồi lấy `len(a)` làm tiêu tốn bộ nhớ, ta chỉ cần 1 biến đếm cộng dồn là đủ. Python cho phép viết Generator Expression kết hợp hàm `sum()` để đếm trong đúng 1 dòng.

```python
import math

def checksonguyento(n):
    if n < 2: return False
    for i in range(2, math.isqrt(n) + 1):
        if n % i == 0: return False
    return True

t = int(input())
for _ in range(t):
    n = int(input())
    # Tính tổng những số i sao cho gcd(i, n) == 1 (True = 1, False = 0)
    count = sum(1 for i in range(1, n + 1) if math.gcd(i, n) == 1)
    
    if checksonguyento(count):
        print("YES")
    else:
        print("NO")
```
