# PY01056 - Xâu thăng bằng

## Đề bài
Một xâu ký tự được gọi là **thăng bằng** nếu khoảng cách của tất cả các cặp ký tự cạnh nhau trong xâu đó đều bằng với khoảng cách của 2 vị trí tương ứng trong xâu đảo của nó.
Công thức toán học: `|s[i] - s[i+1]| == |s_dao[i] - s_dao[i+1]|` với mọi vị trí `i`.

## Ý tưởng cốt lõi
Bài toán này kiểm tra năng lực thao tác vòng lặp và xử lý chuỗi:
1. **Tạo xâu đảo (s_dao)**: Dùng vòng lặp chạy ngược từ cuối mảng (`len(s)-1`) về đầu (`0`) để nhặt từng ký tự nhét vào mảng/chuỗi mới.
2. **Kiểm tra song song**: Dùng một vòng lặp `for i in range(len(s) - 1)` để xét từng cặp liền kề.
3. **Tính khoảng cách**: Dùng hàm `ord()` để lấy thứ tự trên bảng mã ASCII, sau đó dùng hàm `abs()` lấy trị tuyệt đối của phép trừ.
4. **Cắm cờ (Flag)**: Tin tưởng xâu là thăng bằng từ đầu (`check = True`). Hễ có 1 cặp không bằng nhau, lập tức gán `check = False` và ngắt vòng lặp (`break`).

---

## Pseudocode
```text
Lặp số_lượng_test lần:
    s = Nhận_chuỗi()
    y = Mảng_rỗng_chứa_xâu_đảo()
    check = ĐÚNG
    
    # Bước 1: Tạo xâu đảo bằng tư duy vòng lặp cơ bản
    Lặp i từ (Độ_dài(s) - 1) lùi về 0:
        Thêm s[i] vào mảng y
        
    # Bước 2: So sánh khoảng cách
    Lặp i từ 0 đến (Độ_dài(s) - 2):
        khoang_cach_s = Tuyệt_đối( Mã_ASCII(s[i]) - Mã_ASCII(s[i+1]) )
        khoang_cach_y = Tuyệt_đối( Mã_ASCII(y[i]) - Mã_ASCII(y[i+1]) )
        
        Nếu khoang_cach_s khác khoang_cach_y:
            check = SAI
            Dừng Lặp (break)
            
    Nếu check là ĐÚNG:
        In ra "YES"
    Ngược lại:
        In ra "NO"
```

## Giải pháp thực tế (Dựa trên logic vững chắc)
Đoạn mã lột tả tư duy mảng thuần túy, không màu mè "trick" Python:

```python
n = int(input())
for _ in range(n):
    s = input().strip()
    y = [] # Lưu xâu đảo
    check = True
    
    # Tạo xâu đảo
    for i in range(len(s)-1, -1, -1):
        y.append(s[i])
        
    # Duyệt và đối chiếu
    for i in range(len(s)-1):
        if abs(ord(s[i]) - ord(s[i+1])) != abs(ord(y[i]) - ord(y[i+1])):
            check = False
            break
            
    if check:
        print("YES")
    else:
        print("NO")
```

## Pythonic Way (Tuyệt kỹ ăn gian dòng code)
Khi đã hiểu rõ bản chất, ta hoàn toàn có thể dùng Slicing `[::-1]` để đảo chuỗi trong nửa nốt nhạc:

```python
n = int(input())
for _ in range(n):
    s = input().strip()
    s_dao = s[::-1]
    check = True
    
    for i in range(len(s)-1):
        if abs(ord(s[i]) - ord(s[i+1])) != abs(ord(s_dao[i]) - ord(s_dao[i+1])):
            check = False
            break
            
    if check:
        print("YES")
    else:
        print("NO")
```
