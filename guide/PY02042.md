# PY02042 - Hiệu số nguyên lớn

## Đề bài
Cho hai xâu ký tự A và B mô tả hai số nguyên dương lớn. Số lượng chữ số có thể rất lớn (lên tới hàng nghìn chữ số).
Hãy tính hiệu $A - B$.
*(Giống hệt bài PY02041, chỉ thay phép cộng bằng phép trừ).*

## Ý tưởng cốt lõi
1. Đọc hai chuỗi A và B từ Input theo từng dòng.
2. Ép kiểu trực tiếp hai chuỗi này sang số nguyên bằng `int()`.
3. Lấy số thứ nhất trừ đi số thứ hai rồi in ra màn hình.

*(Lưu ý: Tương tự như bài Tổng số nguyên lớn, ta tiếp tục lạm dụng cơ chế **BigInt** (số nguyên độ dài vô hạn) cực kỳ mạnh mẽ của lõi Python để giải quyết bài toán C++ hóc búa này trong vòng một nốt nhạc).*

## Pythonic Way (Phương pháp tối ưu)
Code của Chủ nhân đã rất sát với tinh thần Python rồi. Ta hoàn toàn có thể rút gọn thành 1 dòng tương tự như bài PY02041.

```python
print(int(input()) - int(input()))
```

**Ưu điểm:**
- Vượt ải mọi giới hạn bộ nhớ của máy tính. 
- Ngắn gọn, không lưu trữ biến thừa.
- Code một dòng trông rất "dằn mặt" những ai đang còng lưng viết 50 dòng code xử lý mảng bằng C++.
